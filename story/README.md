Afterlight — Story Survival (Demo)

Files in this folder:
- index.html   — main HTML file
- styles.css   — styles and animated UI
- script.js    — game logic

How to run
1. Open the "story/index.html" file in a modern browser (Chrome, Firefox, Edge, Safari).
2. Click "New Game" and make choices each day. Survive as many days as you can.

Notes
- The game uses localStorage to remember your best run under the key `afterlight_best`.
- Click Save to save a full run to localStorage; Load to restore it.
- This is a small demo: you can extend it with more events, art, audio and mobile controls.

Enjoy!
