Mini Pac-Man Demo

Files:
- index.html — main page
- styles.css — UI styles
- script.js — game logic

How to run:
1. Open index.html in a modern browser.
2. Click "Start Game". Use arrow keys or WASD to move.

Notes:
- High score stored in localStorage under key "pac_best".
- Toggle sound with the button in the HUD.

This demo is a compact Pac-Man-like game for demonstration and learning.
